## Project Overview

This project provides a comprehensive historical analysis of the **Ballon d'Or** award winners, covering the period from 1956 up to 2025 (including historical data and future projections/hypothetical data).

The analysis is presented through an interactive dashboard built using **Microsoft Power BI**, designed to explore key trends such as the distribution of winners by Nationality, Club, and the frequency of wins across different eras.

## Repository Contents

* `ballondor_winners.xlsx - Sheet1.csv`: The raw data source file containing the winners' table (Year, Player, Nationality, Club).
* `ballondor.pbix`: The Power BI project file, which includes the Data Model, Measures, and all Visualizations.
* `Overview.png`: A quick screenshot preview of the final Power BI dashboard.

## üìä Key Insights

The interactive Power BI dashboard provides insights on:

1.  **Top Winners:** Analysis of players with the most Ballon d'Or awards.
2.  **Club Success:** Ranking of clubs with the highest number of award-winning players.
3.  **National Dominance:** Distribution of winners based on their nationality.
4.  **Temporal Evolution:** Tracking the changes in winners and their affiliations across the years.
